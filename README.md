### How to Install

Open the terminal and type following commands.

* ```apt update```
* ```apt install git php openssh curl -y```
* ```git clone https://github.com/rixon-cochi/tech-hack.git```
* ```cd tech-hack```
* ```chmod +x *```
* ```unzip sites.zip```
* ```bash cracker.sh```
